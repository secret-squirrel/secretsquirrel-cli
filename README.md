# Secret Squirrel - Console client

## Getting Started

```
git clone git@github.com:twg/secretsquirrel-cli.git && cd secretsquirrel-cli
npm install
node squirrel.js
```

## Tests

`npm test`

Or install mocha globally

```
npm install -g mocha
mocha
```
